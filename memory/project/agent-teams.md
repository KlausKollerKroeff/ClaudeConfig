# Agent Teams

- **Configuration:** Requires environment variable `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1`
- **Documentation:** https://code.claude.com/docs/en/agent-teams
- **Hiring Flow:** Requires running a terminal command in Claude Code to finalize hiring.
- **UI Preference:** Requested longer popup durations for agent messages.
- **Manager Agent:** Created "Trevor" to act as primary coordinator; distributes tasks to sub-agents.
- **Workflow:** User's main focus is that Trevor is woken up automatically when Claude is called for code-related actions, then orchestrates sub-agents to execute tasks.
- **Integration:** User seeks to connect localhost agent setup with Claude Code agent teams workflow.
