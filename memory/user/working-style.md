# Working Style

- **Initial Execution:** Prefers to run new tools, scripts, or setups manually on the first attempt to verify behavior before enabling automation.
- **CLI Interaction:** Experiences friction with non-responsive prompts; Enter key sometimes fails to trigger acceptance/confirmation during skill setup.
- **Permissions:** Grants permissions automatically when creating or hiring new agents.
- **AI Identity:** Refers to the assistant as "mateu" or "mateo" in conversation.
- **Feedback:** Avoid self-sabotaging actions (e.g., killing own background sessions). "Shooting your own foot" is a direct correction.
- **API Preference:** Prefers OpenRouter over Claude API for agent execution; explicitly refused Claude API changes.
- **Folder Trust:** Wants automatic folder trust when opening agent terminals to avoid manual confirmation prompts.
- **Agent Naming:** Expects agent names to follow proper capitalization (e.g., "Ellie" not "ellie").
- **When to apply:** During initial configuration, testing, or when introducing unfamiliar tools.
